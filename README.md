# WelearnTimeFinsh
WeLearn刷学习时长

### 特别感谢Avenshy提供的v0.1 和 v0.2

## 简介
* 输入账号密码一键学习
* 可以自行选择课程
* 多线程学习，同时学习整个单元
* 可以指定时间或设置随机时间
